# Demo Script

1. Open `/web/` in mock mode and show separate Market Regime and Yield Decay monitors.
2. Run **Safe Yield Allocation** in Execution Proof Arena; explain `ALLOW`.
3. Trigger a volatility scenario and run **Volatile Restriction**; explain tighter limits.
4. Trigger yield decay and run **Yield Decay Block**; explain factor scores.
5. Trigger panic and run **Panic Block**; explain risk-increasing action rejection.
6. Run **Safe Unwind**; explain why withdrawal/repay remains available.
7. Switch to Web3 mode. Connect a Pharos Atlantic wallet and call `checkAction`.
8. Execute only after explicit wallet confirmation and show the explorer receipt.

Never claim live proof without a real Atlantic report ID, transaction hash or deterministic revert, and block number.
