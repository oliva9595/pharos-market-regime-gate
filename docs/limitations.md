# Limitations

- Atlantic V2 has not been deployed from this workspace as of June 13, 2026.
- Dashboard deterministic scenarios are demonstrations, not live market feeds.
- Public V1 addresses remain configured until V2 deployment is completed.
- SlippageGuard supports a limited set of Uniswap V2-style routes.
- Reporter correctness depends on source quality and key security.
- Owner and reporter roles are not yet multisig-controlled.
- Historical analytics use bounded in-process storage rather than a production database.
- The system is testnet middleware, has not received a formal audit, and is not financial advice.
